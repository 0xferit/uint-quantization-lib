uint-quantization-lib

v7.1.0

Ethereum storage charges 20,000 gas per cold write to a 256-bit slot, even when values use a fraction of the bits available. This library shrinks values by discarding low-order bits (right-shift), packs the result into a smaller type, and restores it by shifting back. The tradeoff: controlled precision loss for cheaper, denser storage.
